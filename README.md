# 🏦 Loan Risk Prediction System (Advanced)

## 📌 Overview

This project is an end-to-end machine learning system that predicts loan approval based on applicant details. It includes data preprocessing, model training, and a user-friendly web interface built with Streamlit.

---

## 🚀 Features

* Full loan application form (real-world inputs)
* Machine learning model (Random Forest)
* Automatic handling of categorical variables
* Prediction with confidence score
* Interactive Streamlit UI

---

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn
* Streamlit

---

## ▶️ How to Run

### 1. Install dependencies

pip install pandas scikit-learn streamlit

### 2. Run the app

streamlit run app.py

---

## 📊 Model Details

* Algorithm: Random Forest Classifier
* Handles categorical encoding using one-hot encoding
* Trained on real loan dataset

---

## 💡 Use Case

This system helps financial institutions assess loan eligibility based on applicant risk profile.

---

## 🧠 Key Concept

The project demonstrates how machine learning models can be integrated into real-world applications with proper preprocessing and user interfaces.
